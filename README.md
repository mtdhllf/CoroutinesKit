# CoroutinesKit

安卓协程库,支持UI/非UI线程调度,定时延迟任务,心跳任务及超时操作

已实现能力如下:

- [x] 主线程切换
- [x] IO线程切换
- [x] 支持Handler方式
- [x] 协程方式
- [x] 心跳任务(间隔及时间)
- [x] 倒计时
- [x] 任务取消
- [ ] 超时中断
